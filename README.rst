Popo-reader
===========

A very basic script to seek for every font and texts from a PSD file.

Requirements
************

* Python >= 3.5;
* python-env;
* `psd-tools2`_;
* Pillow;

Install
*******

::

    git clone https://github.com/sveetch/Popo-reader.git
    cd Popo-reader
    make install

Usage
*****

::

    source venv/bin/activate
    python seeker.py Some-PSD-file.psd
